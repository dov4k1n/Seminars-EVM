Differential Equations

Существование и единственность решений

Постановка задачи Коши

Пусть задано \( n \)-мерное уравнение

$$ \dot{x} = f(t,x), \ \ \ \ \ (t,x) \in G, \ \ \ \ \ G \subseteq \mathbb{R}^{1+n}, \ \ \ \ \ f:G\to\mathbb{R}^n \ \ \ \ \ \ \ \ \ \ (\ 1\ ) $$
Задача Коши получается добавлением к уравнению ( 1 ) начального условия
$$ x(t_0) = x_0 \ \ \ \ \ ((t_0, x_0) \in G) \ \ \ \ \ \ \ \ \ \ (\ 2\ ) $$
причём используется следующая терминология:
  • функция \(f\) называется правой частью уравнения ( 1 )
  • начальная точка \( (t_0, x_0) \) состоит из начального момента \(t_0\) и начального значения \(x_0\)
  • решение задачи Коши – это функция \( \mathsf{x}: I \to \mathbb{R}^n \), удовлетворяющая условиям ( 1 ) и ( 2 ), если совершить в них подстановку \( x = \mathsf{x}(t) \). То есть функция, удовлетворяющая равенствам
    $$ \dot{\mathsf{x}} = f(t,\mathsf{x}(t)), \ \ \ \ \ t \in I, \ \ \ \ \ \mathsf{x}(t_0) = x_0 $$
  • переменная \( x \in \mathbb{R}^n \) называется фазой (или фазовой переменной), а \( t \in \mathbb{R} \) – временем


Если в \( \mathbb{R}^n \) фиксирован базис, то одно уравнение ( 1 ) записывается покоординатно в виде системы

$$ \begin{cases} \dot{x}_1 = f_1(t,x_1,..., x_n) \\ ... \\ \dot{x}_n = f_n(t,x_1,..., x_n) \end{cases} $$
а начальное условие ( 2 ) – в виде
$$ \begin{cases} \dot{x}_1(t_0) = (x_0)_1 \\ ... \\ \dot{x}_n(t_0) = (x_0)_n \end{cases} $$
Поэтому само уравнение ( 1 ) нередко называют нормальной системой, состоящей из \( n \) дифференциальных уравнений первого порядка. "Нормальной" – то есть разрешённой относительно производной \( \dot{x} = (\dot{x}_1,...,\dot{x}_n) \)


Решение \( \mathsf{x} \):

  • является продолжением решения \( \mathsf{y} \), если выполнено равенство
    $$ \mathsf{y} = \mathsf{x}\vert _{D(\mathsf{y})} $$
  • непродолжаемо, если оно не имеет продолжений, отличных от самого себя




Локальная теорема существования и единственности

Если \( f, f'_x \in C(G) \), то для любой точки \( (t_0, x_0) \in G \) в некоторой окрестности \( U(t_0) \) точки \( t_0 \) существует единственное решение

$$ \mathsf{x}: U(t_0) \to \mathbb{R}^n $$
задачи Коши ( 1 ), ( 2 ).

Условие \( f, f'_x \in C(G) \), равносильное условию

$$ f,\ \frac{\partial f^i}{\partial x^j} \in C(G), \ \ \ \ \ i, j = 1, ..., n $$
можно заменить:
  • более сильным, но легче проверяемым условием \( f \in C^1(G) \), слегка ослабив тем самым теорему
  • (Теорема Пикара) более слабым, но труднее проверяемым требованием – непрерывностью функции \( f \) по \( t \) вместе с её липшицевостью по \( x \), обозначаемой \( f \in Lip_x(G) \) и подразумевающей, что некоторая константа \( L \) осуществляет оценку
    $$ \left\lvert f(t,y) - f(t,x) \right\rvert \leqslant L\left\lvert y - x\right\rvert, \ \ \ \ \ (t,x),\ (t,y) \in G $$


Теорема Пеано

Если \( f \in C(G) \), то для любой точки \( (t_0, x_0) \in G \) существует решение задачи Коши ( 1 ), ( 2 )


Теорема существования непродолжаемого решения

Любое решение любого уравнения ( 1 ) можно продолжить до непродолжаемого


Теорема единственности непродолжаемого решения

Если для уравнения ( 1 ) все точки области \( G \) – точки существования и единственности, то для любой точки \( (t_0, x_0) \in G \) существует единственное непродолжаемое решение задачи Коши ( 1 ), ( 2 )
